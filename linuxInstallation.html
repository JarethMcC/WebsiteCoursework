Instructions to Install FIFA Transfer Calculator for Linux

Install latest version of Python
  1. First check to see if you have Python installed using the command "python3 --version", if you have Python already installed then move to executing the program
  2. If you do not have Python already installed from the command line use the command "sudo apt-get install python3.8"
Executing the Program
  1. First find where your program is downloaded to and make sure that the player .txt file is in the same directory
  2. From the command line navigate to the folder where the .py file is located
  3. Make sure the player file .txt is also in the same directory before running the program
  4. Use the command "python3 TransferCalculatorV1.4"(Or whatever version you have download)
Enjoy using the FIFA Transfer Calculator!
