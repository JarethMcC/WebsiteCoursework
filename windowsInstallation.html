Instructions to Install FIFA Transfer Calculator for Windows 10

Install latest version of Python
  1. Open the Windows App Store
  2. Use the search bar to find "Python 3.8" or later
  3. Click "Get" to install
Execute Program Through Command Prompt
  1. First find where your program is download to and make sure that the player .txt file is in the same folder
  2. Open Command Prompt and navigate to the folder where the .py file is located
  3. Make sure the player file .txt is also in the same folder before running the program
  4. Use the command "python3 TransferCalculatorV1.4"(Or whatever version you have download)
Enjoy using the FIFA Transfer Calculator!
